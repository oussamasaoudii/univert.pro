## خطوات التثبيت والاستخدام

### الخطوة 1: تشغيل السكريبتات الأساسية

قبل إضافة حسابات الاختبار، تأكد من تشغيل السكريبتات الأساسية:

```bash
# 1. نقاط النهاية الأساسية
mysql -u root -p your_database < scripts/000_core_schema.sql

# 2. البيانات الأولية
mysql -u root -p your_database < scripts/010_seed_initial_data.sql

# ... (السكريبتات الأخرى)

# 3. الدول والأسعار
mysql -u root -p your_database < scripts/030_countries_and_pricing.sql
```

### الخطوة 2: إضافة حسابات الاختبار

```bash
mysql -u root -p your_database < scripts/999_seed_test_data.sql
```

### الخطوة 3: تشغيل التطبيق

```bash
npm run dev
```

ثم اذهب إلى: `http://localhost:3000/auth/login`

## ميزات الحسابات

### حساب المستخدم العادي
- إمكانية الوصول إلى لوحة التحكم العامة
- عرض الخطط والأسعار
- إمكانية الاشتراك في الخدمات
- إدارة الملف الشخصي والإعدادات

### حساب المسؤول
- إمكانية الوصول إلى لوحة التحكم الإدارية
- إدارة المستخدمين والدول
- إدارة الأسعار والخطط
- عرض التقارير والإحصائيات

## إنشاء كلمات سر جديدة (Advanced)

إذا أردت تغيير كلمات السر أو إضافة حسابات جديدة:

### استخدام bcryptjs

```javascript
const bcryptjs = require('bcryptjs');

// توليد hash جديد
const password = 'MyNewPassword123';
const hash = bcryptjs.hashSync(password, 10);

console.log(hash);
// استخدم الـ hash الناتج في السكريبت SQL
```

### تحديث السكريبت SQL

1. استبدل كلمة السر وكمان hash في `scripts/999_seed_test_data.sql`
2. قم بتشغيل السكريبت مرة أخرى

## نصائح الأمان

⚠️ **تنبيه أمان هام:**

- **لا تستخدم هذه الحسابات في الإنتاج** - استخدمها في التطوير فقط
- غير كلمات السر قبل نشر التطبيق
- استخدم متغيرات البيئة لتخزين بيانات المسؤولين الفعليين
- لا تحفظ كلمات السر في ملفات النصوص

## استكشاف الأخطاء

### الخطأ: "Invalid credentials"
- تأكد من تشغيل سكريبت إنشاء الحسابات
- تأكد من أن قاعدة البيانات تعمل بشكل صحيح

### الخطأ: "Table not found"
- تشغيل السكريبتات بالترتيب الصحيح
- التحقق من أن السكريبتات الأساسية نفذت بنجاح

### لا تظهر بيانات الأسعار
- تأكد من تشغيل `030_countries_and_pricing.sql`
- تحقق من أن البيانات موجودة: `SELECT * FROM country_plan_prices;`

## خطوات إضافية

### 1. إضافة صورة الملف الشخصي
بعد تسجيل الدخول، اذهب إلى:
- الإعدادات → الملف الشخصي → رفع صورة

### 2. اختبار الشراء
- اذهب إلى صفحة الأسعار
- اختر دولة
- اختر خطة
- انقر على "ابدأ الآن"

### 3. اختبار لوحة التحكم الإدارية
- قم بتسجيل الدخول كمسؤول
- اذهب إلى `/admin/countries`
- اذهب إلى `/admin/pricing`
